# Color Palette Reference

## Primary Colors (Indigo Theme)
- **Primary**: #6366F1 - Main app color
- **Primary Dark**: #4F46E5 - Darker variant for pressed states
- **Secondary**: #06B6D4 - Cyan accent
- **Tertiary**: #EC4899 - Pink highlight
- **Accent**: #F59E0B - Amber warning

## Neutral Colors (Gray Scale)
- **White**: #FFFFFF - Backgrounds and text on dark
- **Black**: #1F2937 - Main text color
- **Gray 50**: #F9FAFB - Lightest background
- **Gray 100**: #F3F4F6 - Light background
- **Gray 200**: #E5E7EB - Borders and dividers
- **Gray 300**: #D1D5DB - Subtle elements
- **Gray 400**: #9CA3AF - Placeholder text
- **Gray 500**: #6B7280 - Secondary text
- **Gray 600**: #4B5563 - Body text
- **Gray 700**: #374151 - Dark text

## Status Colors
- **Success**: #10B981 - Green for positive actions
- **Warning**: #F59E0B - Amber for caution
- **Error**: #EF4444 - Red for errors
- **Info**: #3B82F6 - Blue for information

## Spacing System
- **XS**: 4dp - Minimal spacing
- **SM**: 8dp - Small gaps
- **MD**: 12dp - Standard spacing
- **LG**: 16dp - Content padding
- **XL**: 20dp - Section margins
- **XXL**: 24dp - Header padding

## Corner Radius
- **Small**: 8dp - Input fields, small components
- **Medium**: 12dp - Cards, buttons
- **Large**: 16dp - Major cards, headers
- **XL**: 20dp - Large containers

## Typography Sizes
- **Display**: 32sp - Main headings
- **Headline**: 28sp - Section titles
- **Title**: 18sp - Card titles
- **Subtitle**: 16sp - Subtitles
- **Body**: 14sp - Main text
- **Label**: 12sp - Labels, badges
- **Caption**: 11sp - Secondary text, dates

## Elevation Levels
- **Low**: 2dp - Subtle elevation
- **Medium**: 4dp - Standard cards
- **High**: 8dp - Prominent elements
